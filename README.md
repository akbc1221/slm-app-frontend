## SLM predictor app ##
<hr/>
<p>This app helps predict the quality of the Selective Laser Modeling output from given set of parameters using Machine Learning model </p>

### Technologies used ###
- **React.js**
- **Flask python framework**
- **SQLite Database**
- **Machine Learning packages**
- **Git VCS**
